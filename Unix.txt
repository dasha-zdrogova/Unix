1)Команда kill необходима для завершения процессов.
Если вызывать команду kill без ключа, то по умолчанию команда kill будет посылать процессу сигнал SIGTERM. Этот сигнал укажет процессу о том, что необходимо завершиться, однако процесс при этом может остаться открытым
Если вызывать команду kill  с ключом 9, то команда kill будет посылать процессу сигнал SIGKILL, в этом случае сигнал не может быть проигнорирован и процесс будет завершён

-Проверить, есть ли файл в текущем каталоге можно с помощью 

if [ -f "file.sh" ]; then
    statements
fi

-Установить владельцем файла пользователя user группы users можно с помощью
$ sudo chown user "путь к файлу file.sh"

- Установить владельцу право на чтение, запись и исполнение
- Установить группе право на чтение и исполнение
- Остальным закрыть доступ к файлу
$ chmod 750 "путь к файлу file.sh"
